fufufafa-shell is a smart and user-friendly command line shell. For more information, visit fufufafashell.com.

fufufafa-shell will be installed into /usr/local/, and its path will be added to /etc/shells if necessary.

Your default shell will not be changed. To make fufufafa-shell your login shell after the installation, run:

chsh -s /usr/local/bin/fufufafa-shell

Enjoy! Bugs can be reported on GitHub.
